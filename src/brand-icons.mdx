# Brand icons

We're recommend to use [FontAwesome 5 for React](https://github.com/FortAwesome/react-fontawesome) to show high quality SVG brand icons.

```jsx
// FontAwesome Facebook icon
import { faFacebook } from '@fortawesome/free-brands-svg-icons'
// React goodshare.js components
import { FacebookButton } from "react-goodshare-components";

<FacebookButton icon={faFacebook} />
```
