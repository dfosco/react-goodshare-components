# Facebook

Provider component for Facebook social network.

* **Full component**, include styles, share link and counter
* **Only share link**, include only share link without any styles
* **Only share counter**, include only share counter without any styles

## Full styled component

This is all-in-one component. Included some awesome styles, share link and counter. We are strongly recommend you to use it for quick prototyping (or else), instead separated components.

<Playground>
    <FacebookFullButton />
</Playground>
